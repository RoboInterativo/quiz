===========
Install
============
Firs of all, please install virtualenv

 ./preinstall_sudo.sh

This step need sudo rights. Usually  hosting with python include virtualenv
and you can skip this step.
Next install requirements:

 ./preinstall_sudo.sh

=========
Run bot
=========

Export telegram token and run
Для запуска импортируйте переменную с telegram и запустите бота коммандой:

 export TOKEN=TOKEN_VALUE
 ./run.sh
